# 🌍✨ Atrativos Turísticos - AVARÉ ✨🌍

## 📖 Sobre o projeto
Este projeto em HTML e CSS apresenta um site fictício sobre os principais atrativos turísticos da cidade de Avaré (SP).
O site está dividido em dois containers principais, cada um contendo três atrativos:

- **Horto Florestal**
- **Represa de Jurumirim**
- **Santuário Nossa Senhora das Dores**
- **EMAPA (Exposição Municipal Agropecuária de Avaré)**
- **Cristo Redentor (Mirante)**
- **Mercado Municipal**

Cada atrativo possui:
- Título e imagem ilustrativa
- Descrição explicativa
- Botão “Saiba Mais”
- Referência da imagem utilizada

O layout foi desenvolvido utilizando W3.CSS para responsividade e estilização, além de CSS externo para alguns ajustes.

## 💻 Linguagens utilizadas
- HTML5
- CSS interno
- W3.CSS

## ⚙️ Estruturas utilizadas
- **h1, h2, h3:** para títulos e subtítulos.
- **img:** para exibir imagens dos atrativos.
- **p:** para textos explicativos.
- **div e classes/IDs:** para organização e estilização.
- **Botões:** para interação “Saiba Mais”.
