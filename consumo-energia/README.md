# ⚡ Consumo de Energia ⚡
## 📖 Sobre o projeto
Este sistema calcula o consumo mensal de energia elétrica de um aparelho doméstico.
O objetivo é praticar lógica de programação e manipulação de entradas em Python.

## Linguagem utilizada
![Python](https://img.shields.io/badge/Python-3.14-blue?style=plastic&logo=python&logoColor=3776AB&labelColor=white&color=3776AB)

## ⚙️ Fórmula utilizada
Consumo mensal (kWh) = (Potência (W) × Horas/dia × Dias/mês) ÷ 1000

- **Potência (W):** potência do aparelho em watts  
- **Horas/dia:** tempo médio de uso diário  
- **Dias/mês:** número de dias de uso no mês  
- Divisão por 1000 → converte de Wh para kWh
