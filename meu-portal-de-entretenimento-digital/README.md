# 🎬🎶🎮 Meu Portal de Entretenimento Digital 🎮🎶🎬

## 📖 Sobre o projeto
Este projeto em HTML e CSS apresenta um portal de entretenimento digital dividido em três seções principais:
- **Filmes e Séries**
- **Música e Podcasts**
- **Games e Tecnologia**

Cada seção contém:
- **Um título**
- **Três itens com imagem ilustrativa**
- **Pequenos textos explicativos**

O layout foi desenvolvido utilizando **Flexbox**, garantindo organização em colunas. Foram aplicados os três tipos de CSS (inline, interno e externo), além de classes e IDs para padronização dos estilos.

## 💻 Linguagens utilizadas
- HTML5  
- CSS3

## ⚙️ Estruturas utilizadas
- **h1, h2, h3:** para títulos e subtítulos.  
- **img:** para exibir imagens ilustrativas.  
- **p:** para textos explicativos.  
- **div e classes/IDs:** para organização e estilização.  
- **Flexbox:** para organizar os itens em colunas.  
- **CSS inline, interno e externo:** aplicados em diferentes partes da página.
