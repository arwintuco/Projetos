# 🌐 Meus Sites Favoritos 🌐
## 📖 Sobre o projeto
Este projeto em HTML apresenta uma página simples listando meus sites favoritos.
Cada site é exibido com:
- Um título
- Uma imagem ilustrativa
- Um link clicável para acessar o site diretamente

Os sites destacados são:
YouTube Music, Archive of Our Own (AO3), GitHub, YouTube e IBM

## 💻 Linguagem utilizada
HTML5

## ⚙️ Estruturas utilizadas
- **h1, h3:** para títulos e subtítulos.
- **img:** para exibir imagens representando cada site.
- **a:** para criar links clicáveis que levam ao site correspondente.
- **br:** para quebra de linha e melhor organização visual.
