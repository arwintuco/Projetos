# 🐾💙 ClinVet - Cuidando com Amor e Ciência 💙🐾

## 📖 Sobre o projeto
Este projeto em HTML e CSS apresenta o site fictício da clínica veterinária ClinVet, dividido em três seções principais:
- **Home** (História, Missão, Visão e Valores)
- **Parceiros** (empresas e organizações colaboradoras)
- **FAQ** (perguntas frequentes e formulário de contato)

Cada seção contém:
- Um cabeçalho com logo e slogan
- Conteúdos organizados em listas e blocos
- Rodapé com informações de contato e redes sociais

O layout foi desenvolvido utilizando W3.CSS para estilização responsiva, além de **fontes externas** e **ícones do Font Awesome**.

## 💻 Linguagens utilizadas
- HTML5
- W3.CSS
- CSS interno

## ⚙️ Estruturas utilizadas
- **h1, h2, h3, h4:** para títulos e subtítulos.
- **img:** para exibir logotipos e imagens ilustrativas.
- **p:** para textos explicativos.
- **ul e li:** para listas de valores e informações.
- **div e classes/IDs:** para organização e estilização.
- **Flexbox e W3.CSS:** para responsividade e alinhamento dos elementos.
